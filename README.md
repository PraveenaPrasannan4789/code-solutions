# LeetCode 1-Month Preparation Plan (JavaScript)

This repository contains my structured 1-month preparation plan for coding interviews using JavaScript.  
The focus is on building strong problem solving skills using common data structure and algorithm patterns.

---

## Goal

- Solve 3 problems per day  
- Focus on patterns instead of memorising  
- Improve coding speed and accuracy  
- Prepare for full stack developer interviews  

---

Focus on HashMap, Set, and array fundamentals
